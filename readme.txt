1.运行本目录例程前，请安装对应日期runtime插件。
2.Media文件夹存放着例程中所需要的所有数据。
3.例程均通过相对路径去获取Media文件夹中所需要的数据，为保证例程的正常运行，请不要随意更改父目录（Samples）和Media文件夹的名称。
4.每个例程在运行exe时，按F1键可打开对应的“操作指南”。
5.使用community.FDB的例程：
  FeatureLayerValueMapRender  使用层高、楼层数等属性字段
  FeatureClassQuery、FeatureCreateAndDelete、FeatureEdit  展示通过解析逻辑树获取GroupId对应的GroupName的过程
  FeatureLayerSimpleRender  对IGeometryRender::RenderGroupField的运用，展示解析逻辑树xml设置FeatureLayer可见
